# 校园流浪猫信息平台

一个用于展示和管理校园流浪猫信息的网站，支持图片和文字信息展示，帮助师生更好地了解和关爱校园流浪猫。

## 功能特性

### 🐱 核心功能
- **猫咪信息展示**: 展示每只猫咪的详细信息，包括照片、名字、状态等
- **搜索功能**: 根据猫咪名称、特征或描述进行搜索
- **筛选功能**: 按状态筛选猫咪信息
- **响应式设计**: 适配各种设备屏幕尺寸
- **模态框详情**: 点击猫咪卡片查看详细信息

### 🎨 界面特色
- 现代化UI设计，色彩搭配舒适
- 流畅的动画效果和交互体验
- 卡片式布局，信息展示清晰
- 响应式设计，支持手机、平板、电脑

### 🛠️ 技术特性
- 纯前端实现，无需后端支持
- 使用原生HTML、CSS、JavaScript
- 模块化代码结构，易于维护和扩展

## 快速开始

### 1. 直接打开使用
直接双击 `index.html` 文件在浏览器中打开即可使用。

### 2. 本地服务器运行（推荐）
为了更好的体验，建议使用本地服务器运行：

```bash
# 使用Python3
python -m http.server 8000

# 使用Node.js http-server
npx http-server

# 使用PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 使用说明

### 浏览猫咪信息
1. 首页展示所有猫咪的卡片列表
2. 每张卡片显示猫咪的基本信息和缩略图
3. 点击卡片查看详细信息

### 搜索和筛选
1. **搜索框**: 输入关键词搜索猫咪名称或特征
2. **位置筛选**: 选择特定区域（宿舍区、教学楼等）
3. **状态筛选**: 按健康状态筛选猫咪

### 查看详细信息
点击任意猫咪卡片，弹出模态框显示：
- 大尺寸猫咪图片
- 详细的基本信息（位置、年龄、性别等）
- 状态标签
- 详细描述和特征

## 数据格式

猫咪数据存储在 `script.js` 文件的 `catsData` 数组中，每只猫咪包含以下字段：

```javascript
{
    id: 1,                              // 唯一标识
    name: "小橘",                       // 猫咪名字
    image: "图片URL",                   // 猫咪图片
    status: "健康",                      // 健康状况
    age: "2岁",                         // 大概年龄
    gender: "公猫",                      // 性别
    description: "详细描述...",           // 详细描述
    characteristics: "特征描述"          // 外貌特征
}
```

## 自定义和扩展

### 添加新的猫咪信息
在 `script.js` 文件的 `catsData` 数组中添加新的猫咪对象：

```javascript
{
    id: 7,
    name: "新猫咪",
    image: "新图片URL",
    // ... 其他字段
}
```

### 修改样式
- 修改 `style.css` 文件来自定义颜色、字体、布局等
- 主要样式类：`.cat-card`（猫咪卡片）、`.modal`（模态框）、`.header`（头部）

### 添加新功能
- 在 `script.js` 中添加新的功能函数
- 在 `index.html` 中添加对应的HTML元素
- 在 `style.css` 中添加对应的样式
